# Overview

The ASL Dataset is an object detection dataset of a few common ASL words w/ bounding boxes. It was curated as a resource for a student capstone project at Neumont College of Computer Science. This resource is **free for public use**.

Thank you to SignASL.org for providing many of the photos found in this dataset
https://www.signasl.org/

Dataset Author: Cole Belfry
Last Updated: 10/24/2022

# Use Case

This dataset was used to train an object detection model in combination with the **American Sign Language Letters** dataset for realtime sign language translation.
